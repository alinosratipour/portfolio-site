WHAT I DO

Prototyping

I research the client's business, competitors and customers then using wireframing Tools like adobe XD I'll draft an initial prototype. After I sit down with the client and ensure that everything is covered such as the structure of the site, user experience and responsivity for all devices.

PSD/XD Slicing

I can take your design in any format XD/PSD or Sketch and I slice it in HTML, CSS, and JavaScript with care and great attention to the smallest details. I'll make sure that the code I write is easily readable, reliable and robust.

Frontend development

I can develop a website in plain HTML/CSS and vanilla javascript or use libraries and frameworks like React or bootstrap. When it comes to backend I also have hands-on experience using some of the backend technologies such as express.js and node.js.

Performance Testing

I understand how important is to make sure that the website performs smoothly without any malfunction. optimizing images and minifying CSS and JavaScript and testing across different browsers are the steps I take to reduce website loading time.
